# Day 18: Copy Data to an Azure Blob Storage Container

A Blob container named `xfusion-blob-24050` already exists in the `East US` region under the storage account `xfusionst17600`. Copy the file `/tmp/xfusion.txt` to the Blob container `xfusion-blob-24050`.

### Upload files to blob container

1. First go to `All resources` and locate the `Storage account`

   <img width="1084" height="244" alt="image" src="https://github.com/user-attachments/assets/74f21cf2-2068-43b8-aea8-b348bfcb3e28" />

2. Click on the `Storage account` and select `Storage browser` from navigation pane and choose `Blob containers`.

   <img width="1309" height="513" alt="image" src="https://github.com/user-attachments/assets/790702e2-5504-4a21-a0c1-9d22ac7c4556" />

3. Under `Blob containers`, select the existing container and click on `Upload`.

   <img width="1086" height="311" alt="image" src="https://github.com/user-attachments/assets/e18a09fc-3ba6-4d1b-b4ef-22047d0279f9" />

4. Upload the file and click `Upload`.

   <img width="585" height="439" alt="image" src="https://github.com/user-attachments/assets/a1bffe34-833b-4f53-a0e6-12784371565d" />
